Home

This is a project to create a website for myself. Someone happening across this website may wonder what content they will find, that is something I would like to know too. This website may be slightly content light. In the past it has gone through various facelifts and had a few different shapes and forms. but to this point has had nothing to see.

The web site has formed more of a test place for me to try different things as design and ux design are interests of mine.

I do not know what I content I would like to have on this website. I college I study computer engineering, so it would be sensible to have some of the projects I have worked on visible, in some sort portfolio.  I should get on that.

I also like photography and painting, so having some of my hobbies presented in someway would also be neat.

Should I feel the need to have opinions I could have a blog, I doubt I would have any or at least for long enough to write them down. But having wordpress blog so I can learn to write a theme would be nice.

This site will only use my images and artwork. I will code it myself, I'm familiar with php but would like to use ruby (again to learn it). Now the blog will be wordpress so that is an exception and I will use some resources like jQuery and maybe rails.
